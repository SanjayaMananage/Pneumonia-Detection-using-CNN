# Pneumonia-Detection-using-CNN
This project focuses on classifying chest X-ray images as Pneumonia or Normal using deep learning techniques. The dataset, available on Kaggle (Chest X-Ray Pneumonia Dataset), contains 5,863 JPEG images organized into training, validation, and test sets. Since the images vary in size, they were resized and normalized before model training. The dataset is also imbalanced, with more pneumonia cases than normal cases. To address this issue, I implemented a Convolutional Neural Network (CNN) using multiple strategies, including training on the original data, applying class weights, and incorporating data augmentation techniques. The project evaluates how these approaches improve model performance on imbalanced medical image data.
